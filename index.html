Base Size Scaling

The --base-size token controls the root font-size. Since all values use rem, changing it scales the entire framework.

Adjust Base Size

Current: 100% (1rem = 16px)

Preview Card

This entire card — text, padding, spacing, border-radius — scales with --base-size. All values are defined in rem, so changing the root font size scales everything proportionally.

Badge
:root { --base-size: 100%; /* default: 1rem = 16px */ --base-size: 112.5%; /* larger: 1rem = 18px */ --base-size: 87.5%; /* smaller: 1rem = 14px */ }
